Who am I?

I'm Ahmad Shah. I'm currently specializing in Computer Science at the University of Toronto Scarborough. I've been coding for the better part of 5 years, and I strive to get better and better as time goes on. I don't worry too much about the future, but I do aspire to be a crucial and important member of a software development team some time in the near future. In the mean time I will continue to do my best to become a better programmer and developer, and to not lose sight of what caused me to love CS in the first place.

I am passionate about web-development, and am always looking to learn more about HTML, CSS, and Javascript. I am also currently learning the MEAN stack (MongoDB, ExpressJS, AngularJS, NodeJS) to create full stack web applications. I also enjoy coding in Java and Python, and want to eventually get started on a simple game in Unity.

My passion for Computer Science is only rivaled by my love for hockey (GO LEAFS GO!) and history. I am also very interested in language acquistion and learning. I have learned French to an advanced level, and am currently tackling Spanish. I hope to become a polyglot one day.

Binding of Isaac: Bootlegged

  • Used SOLID design principles and Object Oriented Programming

CSCB20 Course Website
  • Created a course website for one of my university courses using HTML, CSS, JavaScript, JQuery, SQL and Flask
  • Only allowed authorized users to access the website. To this end, the user was allowed to create accounts using HTTP requests and then these accounts were stored in an SQL database
  • Allowed users to create accounts and log in as either a student user or instructor, and displayed different information to the user depending on user type. Instructor users were allowed to edit and input marks of all valid student users, while the student users could view these marks.
  • Used flexbox to implement table-less design
  • Used media queries to make website responsive
  • Created a sticky header and a fixed footer
